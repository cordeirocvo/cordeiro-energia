# 🔗 Acesso à Aplicação Cordeiro Energia

## Link da Aplicação (Vercel)

**URL Principal:**
```
https://cordeiro-energia-54bt-cordeirocvos-projects.vercel.app/
```

## Como Acessar pelo Chrome

1. Abra o **Google Chrome**
2. Cole o endereço acima na barra de endereços e pressione **Enter**
3. Faça login com seu usuário e senha

## Perfis de Acesso

| Perfil | Redirecionamento após login | Pode fazer o quê? |
|--------|----------------------------|-------------------|
| **ADMIN** | `/admin` — Painel Administrativo | Tudo: ver instalações, editar, criar usuários, nova atividade |
| **TV** | `/tv` — Dashboard para SmartTV | Somente visualizar o painel em tela cheia |
| **COMUM** | `/publico` — Formulário de Campo | Registrar novas atividades |

## App Mobile (Android)

O APK para Android está em:
```
cordeiro-energia-app/build/cordeiro-energia.apk
```
Instale diretamente no dispositivo Android habilitando "Fontes desconhecidas" nas configurações.

**iOS:** Instale o app **Expo Go** pela App Store e acesse o link provido.
